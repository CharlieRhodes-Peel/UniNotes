# In [[Set]]s:
- A 1 to 1 relation between two sets
- Each function must contain a domain (X) and a codomain (Y)
	- 1 to many mapping not allowed

### Identity function:
- Where no change happens ($f: X -> X$)

### Notation:
- $f: X -> Y$
- $f ; g : X -> Y = g(f(x))$

## Inverse:
- An inverse will not exist if sets used in a function are not the same size
- A [[Bijective]] function will have an inverse
