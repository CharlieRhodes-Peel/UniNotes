# Represented using: $P(X)$ or $2^X$
Is the [[set]] of all [[Subsets]] of any [[set]] X
- P({0,1}) = {$\emptyset$, {0}, {1}, {0,1}}
- All powersets contain the empty [[set]]
	- PP($\emptyset$) = { $\emptyset$,  {$\emptyset$} }

### The cardinality of a power [[set]] is always $2^{|X|}$
 $\therefore$ if cardinality of [[set]] $\not = 2^k$ for any k, then cannot be a power [[set]]
