# Symbols:
## Conjunction: $\bigwedge$
"It is Friday **and** it is raining"
- Think $\wedge$ for **AND**
## Disjunction: $\bigvee$
"It is Friday **or** it is raining"
- Think $\vee$ for the **Other** one so its **OR**
## Exclusive OR: $\bigoplus$
“**Either** today is Friday **or** it is raining today, **but not both**”.
- See in [[Logic Gates]]
## Implication: $\rightarrow$
"**If** it is Friday **then** is is raining today"
## Biconditional: $\leftrightarrow$
“It is raining today if and only if it is Friday today.”
## Not: $\neg$
"It is not raining today"
- if variable r = is raining. Above statement = $\neg$r

# BIDMAS for logic:
1. $\neg$
2. $\wedge$ & $\vee$                                      ## associates to the left 
3. $\rightarrow$                                            ## associates to the right
