# How to Check:
### $v_1 = \alpha_2 v_2 + \alpha_3 v_3$
Sub in the values for the vectors going down the columns
## Example:
$v_1 = (2,1,1)^T, v_2 = (4,2,1)^T, v_3 = (1,1,0)^T$
- $2 = 4 \alpha_2 + \alpha_3$
- $1 = 2 \alpha_2 + \alpha_3$
- $1 = \alpha_2$
Solve these and if they **DONT** have a solution then they are **INDEPENDANT**
