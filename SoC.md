# System on Chip:
- Conceptually similar to a [[Microcontroller]]
- Integrates most of the components of a computer in a **single chip**
- [[Microcontroller]] SoC usually include radio, co-processors, interface drivers, etc

# [[Microcontroller]] SoC with Radio: 
- A [[Microcontroller]] with a radio in the same package
- Useful for IoT applications

# [[Microprocessor]] SoC
- Common in phones, embedded system (Raspberry Pi) and network equipment
- Starting to appear more in laptops too
- Includes:
	- [[CPU]]
	- [[GPU]]
	- [[Networks]]ing
	- maybe [[RAM]]
