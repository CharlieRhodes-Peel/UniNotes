A service from amazon where you can post tasks to it and people can complete tasks for rewards (like money)
### Options you can set
- You can set how many people should complete the task
- How much you pay someone per task
- How much time someone has per task
- If you should give them payment after task complete (because you can reject someone's tasks)
- And then afterwards the poster of the tasks gets an excel spreadsheet (probably a csv icl) with all the information

### Examples of what people use Mech-Turk for

| Task Type            | Example Question                                                               |
| -------------------- | ------------------------------------------------------------------------------ |
| Image Categorisation | Choose the correct category + image                                            |
| Data Collection      | Give a review for (link) this website                                          |

# [[Incentives]]
**Mason & Watts (2009)**
There were 4 different "treatments" of people for this test. Everyone got a base payment of $0.10 per task. With different bonus separating the treatments by: No bonus pay, low ($0.01 per set), medium ($0.05), high ($0.10). The participants could choose to perform up to 99 tasks total. In the end 611 workers sorted 36,425 image sets

Across the board we see that higher pay *increases output* **BUT** *doesn't* increase *quality*
