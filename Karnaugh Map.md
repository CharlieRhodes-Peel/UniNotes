# Rules of Simplification:
- Groups must NOT contain 0
- All 1's must be in atleast 1 group
- Groups can intersect
- Groups can be horizontal & vertical but NOT diagonal
- Groups must contain $2^n$ cells (1,2,4,8, etc...)
- Each group should be as large as possible
- The simplest solution is one with the least number of groups possible
- Groups can wrap around the The table

- You are looking for the numbers that **DON'T** change inside the groups
- Just add the groups together (OR [[Logic Gates]])
