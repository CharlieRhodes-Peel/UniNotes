A **cheat-sheet** of different short hands used throughout the ELEC module

| Shorthand | COMMAND NAME    | Description                                                                 |
| --------- | --------------- | --------------------------------------------------------------------------- |
| lw        | LOAD (WORD)     | **load** a *WORD* **from** memory **into** a register                       |
| lb        | LOAD (byte)     | **load** a *BYTE* **from** memory **into** a register                       |
| sw        | STORE           | **store** something **into** memory **from** a register                     |
| beq       | BRANCH-IF-EQUAL | **Branch** to a position **if** values at two given registers are **equal** |
