# Made up of:
- Opcode (what is the action)
- Operand (what is the data)
- THEY ARE **READ ONLY**
- They are considered **VECTORS** of bits
	- i.e the order of the bit matters

# Instruction Types (Assembly):
- Data processing:
	- [[ALU]] Stuff: Add, Sub, Multiply, Divide
	- Could be (signed) Integer or Floating Pointer
	- Shift and Rotate instructions
	- Logical Instruction [[Logic Gates]]
- Data movement:
	- Load
	- Store
	- Stuff like that
- Jumping between stuff:
	- Branch (may be conditional)
	- Skip
	- Subroutine
	- Loop
